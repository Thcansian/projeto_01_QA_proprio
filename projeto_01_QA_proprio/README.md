# projeto_01_QA_proprio

## Primeiro projeto de QA próprio

Repositório criado para consolidar meus estudos em **Qualidade de Software (QA)**, 
aplicando conceitos de testes manuais, automatizados, testes de API e testes de performance,
utilizando projetos reais (ex: Bezkoder).

Este projeto funciona como um **repositório base**, onde centralizo meus aprendizados práticos em QA.

---

## Objetivos do projeto

- Praticar testes manuais baseados em cenários reais
- Criar e documentar casos de teste
- Executar testes de API
- Realizar testes automatizados
- Aplicar testes de performance
- Aprender boas práticas de documentação em QA
